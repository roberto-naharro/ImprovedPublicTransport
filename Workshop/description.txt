[h1]IPT Essentials[/h1]

Community revival of [url=https://steamcommunity.com/sharedfiles/filedetails/?id=928128676]Improved Public Transport 2[/url] by [url=https://github.com/bloodypenguin]BloodyPenguin[/url], updated for Cities: Skylines 1.21.x. Own line management well; let specialist mods handle everything else.

[b]► Source code & issues:[/b] [url=https://github.com/roberto-naharro/ImprovedPublicTransport]github.com/roberto-naharro/ImprovedPublicTransport[/url]

[h1]What it does[/h1]

[list]
[*][b]Manual vehicle count[/b]. Set exactly how many vehicles run on a line, bypassing the budget calculation. Toggle manual/budget mode per line at any time.
[*][b]Per-line vehicle type selection[/b]. Pick which vehicle models serve each line. Shows thumbnail previews, Workshop badges, and a 3D preview. Mixed-fleet lines survive reload.
[*][b]Depot selection per line[/b]. Pin a line to a specific depot. Falls back to Auto if the depot is removed.
[*][b]Per-line ticket price[/b]. Set a custom fare (free to type cap, 0.10 steps) with a Reset button. Optional happiness consequence for overcharging (on by default). Writes only the per-line fare. Fully reversible.
[*][b]Copy and paste line settings[/b]. Duplicate vehicle count, colour, vehicle types, ticket price, and depot onto another line in two clicks.
[*][b]Per-stop passenger stats[/b]. Passengers In / Out for current week, last week, and rolling average. Rename stops and browse suggested names from nearby buildings.
[*][b]Per-stop unbunching toggle[/b]. Enable or disable vehicle spacing dwell per stop. Stop circles are tinted [b]red[/b] (on) or [b]green[/b] (off) in the line panel. [i]Apply to all stops[/i] button included. Integrates natively with [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2262054175]Express Bus Services[/url], no bridge mod needed. [b]Do not install ExpressBusServices-IPT2[/b]; it conflicts.
[*][b]Per-vehicle stats[/b]. Passengers, earnings, distance (current/last week/average), plus last-stop boarding and alighting count.
[*][b]Line earnings and costs[/b]. Stats table: total passengers, gross earnings, vehicle maintenance cost, and cost per line (infrastructure split across active lines of the same type). Current week / last week / rolling average.
[*][b]Vehicles in this line[/b]. Live list of all vehicles on a line. Right-click to jump to a vehicle; select and Remove to send chosen vehicles back to the depot.
[*][b]Vehicles queued[/b]. Lists vehicles that have left the depot but not yet reached their first stop.
[*][b]Station stop list[/b]. Clicking a transit station shows all associated stops, each clickable to open its stop panel.
[*][b]Line deletion tool[/b]. Bulk-delete all lines of a given transport type from the mod options panel.
[/list]

[b]Incompatible with[/b] [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3007903394]Transport Lines Manager[/url] and [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3690061052]Improved Public Transport 3[/url] (rival full line managers). Enable only one.

[h1]Design philosophy: composability[/h1]

The Cities: Skylines mod ecosystem already has excellent, dedicated tools for vehicle stats and colours, stop placement, and advanced spacing. IPT Essentials is built to work [i]alongside[/i] those specialists, not to replace them.

We focus on line management: precise vehicle count control, per-line vehicle type selection, depot pinning, per-line ticket price, copy and paste, and granular passenger statistics. For everything else, there is already a better, purpose-built solution. Using them together gives you a cleaner, more stable, and more powerful setup than any single mod can offer.

[h1]Migrating from IPT2[/h1]

Save data is fully backward-compatible. Manual vehicle counts and line settings load automatically. [b]Unsubscribe ExpressBusServices-IPT2[/b] before using; it patches the same code paths and will conflict.
